Java Syntax

Allgemeines


public class Main {
public static void main(String[] args) {
System.out.println ("Hello World")";
}


Jeder Zeile Code in Java, die ausgeführt wird, muss sich in einer Klasse befinden.
In unserem Beispiel haben wir die Klasse Main genannt. Ein Klassenname sollte immer mit einem Großbuchstaben beginnen.

Konstruktor


Ein Konstruktor in Java ist eine spezielle Methode, die zum Initialisieren von Objekten verwendet wird.
Er hat denselben Namen wie die Klasse und kann mehrere Parameter enthalten.

Der Standardkonstruktor einer Klasse Tier würde wie folgt aussehen:

public class Tier {

   private String art;
   private String name;

   public Tier(){

   }
}
